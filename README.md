### Requirement: Python>=3.11
- Use uv package manager (pip install uv)
- cd to folder directory
- enter `uv init`
- enter `uv add -r requirements.txt`
- Refer to project.toml for the dependecies information

.env file should contain:
- GROQ_API_KEY=""
- TAVILY_API_KEY=""
- LANGSMITH_API_KEY=""
- OPENAI_API_KEY=""
- GOOGLE_API_KEY=""
